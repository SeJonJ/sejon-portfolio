# sejon-portfolio

**장세존(Jang Sejon) · Backend Engineer 개인 포트폴리오입니다.**

실시간·분산 시스템과 금융권 전자서명 개발 경험을 담은 이력서형 웹 포트폴리오.

- 🔗 **Live:** https://sejon-portfolio.vercel.app/
- 🎨 터미널 테마 · Vite + React + Tailwind CSS + Framer Motion

## 개발

```bash
npm install
npm run dev      # 로컬 미리보기 (http://localhost:5173)
npm run build    # 프로덕션 빌드
```

- 내용 수정: `src/data.js`
- 디자인 수정: `src/App.jsx`

## 배포

GitHub `main` 브랜치에 push하면 **Vercel이 자동으로 빌드·배포**합니다.
